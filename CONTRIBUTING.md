# Contributing

## How to contribute
- Open an issue describing the change (bug, baseline, metric, dataset improvement)
- Submit a PR with:
  - clear description
  - tests or reproducible run output (where applicable)
  - updated docs if behavior changes

## Prompt contributions
All prompts must comply with:
- `DATA_GOVERNANCE.md`
- `SAFETY_POLICY.md`
- No identifying information

## Style
Prefer small PRs with clear scope.
