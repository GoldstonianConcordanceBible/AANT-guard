# Security

## Reporting
If you discover a security issue (e.g., prompt injection that bypasses safeguards), report it via a GitHub issue labeled **security**.

## Scope
This project is research software and not intended for clinical deployment.
No sensitive user data should be processed or stored.

## Prompt Injection Notes
We document injection attempts and mitigations in `ARCHITECTURE.md` and `TRUST_AND_LIMITATIONS.md`.
